SITE AURA+ - Curso de Aparência, Estilo e Físico

Como usar:
1. Abra index.html no navegador.
2. Clique em um plano para gerar a chave.
3. Use a chave gerada para criar a conta.
4. Após login, o aluno entra em dashboard.html.

Keys demo:
- AURA-SEMANAL-2026
- AURA-PERMANENTE-2026

Importante:
- Este é um front-end estático com simulação de pagamento, login, key, diagnóstico e progresso.
- Para vender de verdade, conecte um backend e um gateway como Mercado Pago, Stripe, Hotmart ou Kiwify.
- Nunca deixe chaves reais no JavaScript público. Valide no servidor.
